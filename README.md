####接下来是他的功能
- 左侧是一些时间预设
- 中间显示剩余的时间 单击可开始或暂停
- 右侧可以自定义倒计时的时间 使用鼠标滚轮控制

####字体与铃声
- 你可以在同级目录添加music.mp3作为结束时提醒的音频文件
倒计时剩余三秒结束时会播放，倒计时结束后五秒暂停
- 默认使用的字体是HarmonyOS Sans SC
